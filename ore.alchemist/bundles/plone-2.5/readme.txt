
 Alchemist 0.2 

 requires zope 2.9.4+ and plone 2.5

 the full path to library directory should be specified in your zope's
 instance zope.conf file as additional 'path' option

 the full path to the zproducts directory should be specified in your
 zope instance's zope.conf file as an additional 'products' option

 additional reading can be found for quickstart usage via examining
 the orgpeople product's readme and quickstart.txt documentation.
